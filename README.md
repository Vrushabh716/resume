A clean and professional resume website built using **HTML, CSS, and JavaScript**.  
This project showcases my academic background, technical skills, projects, and experience in a structured and recruiter-friendly format.

The resume is designed to be lightweight, responsive, and easy to maintain, with an option to **print or save the resume as a PDF** directly from the browser.
